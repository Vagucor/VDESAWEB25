# Pecera digital  

## Descripción del proyecto  
Una **pecera virtual** en la que podemos *conseguir peces*, cuidarlos, criarlos y tenerlo de fondo decorativo. ¿Quieres tener una pantalla dinámica pero relajante mientras estudias o trabajas? Con la pecera virtual puedes tenerlo y además podrás **personalizarlo a tu gusto**.  

> "Una pecera digital no es solo decoración, es interacción y relajación al mismo tiempo."  

## Público objetivo  
El público objetivo sería principalmente **juvenil**, aunque también puede atraer a *amantes de los acuarios, peceras y vida marina*.  

## Funcionalidades principales  
Entre las funcionalidades más destacadas:  
- Pantalla decorativa relajante.  
- Sistema de obtención de peces y objetos.  
- Restricciones según el tipo de pez (agua dulce, salada, fría o tropical).  
- Cuidado de los peces: alimentación y limpieza de la pecera.  

### Pasos de uso  
1. **Configura** tu pecera inicial (tipo de agua, temperatura, decoración).  
2. **Consigue** peces y objetos mediante el sistema de recompensas.  
3. **Cuida** de tus peces alimentándolos y limpiando la pecera.  
4. **Personaliza** el entorno para hacerlo único y relajante.  

## Tecnologías previstas  

### Frontend  
Las tecnologías previstas para el frontend serán:  
- **HTML**  
- **CSS**  
- **JavaScript**  

### Backend  
Las tecnologías previstas para el backend serán:  
- **PHP**  
- **MySQL**  

### Ejemplo de estructura básica en HTML  
```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Pecera Digital</title>
  <link rel="stylesheet" href="styles.css">
  <script src="app.js"></script>
</head>
<body>
  <div id="pecera">
    <!-- Aquí aparecerán los peces -->
  </div>
</body>
</html>
```
