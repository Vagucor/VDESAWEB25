# Pecera digital

## Descripción del proyecto
Una pecera virtual en la que podemos conseguir peces, cuidarlos, criarlos y tenerlo de fondo decorativo. ¿Quieres tener una pantalla dinámica pero relajante mientras estudias o trabajas? Con la pecera virtual puedes tenerlo y además podrás personalizarlo.

## Público objetivo
El público objetivo sería un público juvenil, aunque también puede ser para amantes de las peceras
